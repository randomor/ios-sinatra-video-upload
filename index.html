Ios-sinatra-video-upload

a demo of uploading video from iOS to Sinatra, save it to server or S3

HOWTO:

  1. Clone this repo: $ git clone git@github.com:randomor/ios-sinatra-video-upload.git
  2. Open the Sinatra project: cd sinatra-video-upload
  3. Install gemset: bundle install
  4. run Sinatra: ruby -rubygems video.rb
  5. Open the iOS project: open ../ios-video-upload/VideoUpload.xcodeproj
  6. Build and run on your iPad to upload video

You can also upload the video into S3 by run the s3-version branch of Sinatra project
